Estudio de Caso: Dashboard de Arriendos en la RM

Una herramienta interactiva para el análisis del mercado inmobiliario.

A continuación se presenta el dashboard interactivo. **Nota:** puede tardar hasta 50 segundos en cargar la primera vez, ya que utiliza un servidor gratuito de Render que se "duerme" por inactividad.

1. Objetivo del Proyecto

El objetivo de este proyecto era crear una herramienta intuitiva para que potenciales arrendatarios o inversionistas pudieran entender las tendencias de precios en diferentes comunas de Santiago. El desafío principal era consolidar y limpiar datos de diversas fuentes, manejar información geoespacial y presentar los resultados en una interfaz interactiva y fácil de usar.

2. Proceso y Desafíos Técnicos

  • Limpieza de Datos: Se utilizó la librería Pandas en Python para procesar un archivo de texto con más de 1000 registros, manejando inconsistencias como valores monetarios mixtos (CLP y UF) y múltiples formatos numéricos.
  • Análisis Geoespacial: Se utilizó GeoPandas para cargar, filtrar y normalizar un shapefile de las comunas de Chile, asegurando una unión correcta con los datos de arriendo mediante la estandarización de nombres.
  • Desarrollo del Dashboard: La interfaz fue construida con Dash y Plotly Express, implementando callbacks encadenados para crear una experiencia de filtrado inteligente, donde las opciones de "Baños" se actualizan según los "Dormitorios" seleccionados.
  • Despliegue: La aplicación fue contenerizada con Docker para gestionar las dependencias del sistema (GDAL) y desplegada en la nube a través de Render para su acceso público.
